Data Governance Reference Architecture — Future Fund

4-month engagement · Australia's Sovereign Wealth Fund

Overview

Defined the reference architecture for Data Governance — covering Metadata, Lineage, Quality, and Observability — aligned to a Databricks platform for Australia's sovereign wealth fund.

Scope of Work

  • Developed data governance reference architecture patterns implementing governance by design — covering data security, metadata management using Unity Catalog, and data quality using specialised tools such as Soda

  • Documented operating models relevant to data quality and governance management
